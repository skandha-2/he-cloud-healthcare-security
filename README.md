# Privacy-Preserving Security for Cloud-Based Healthcare Databases

**Research Portfolio by Suresh Kumar Uppada, Ph.D.**

This repository presents a focused research portfolio on homomorphic encryption, cloud-based healthcare databases, privacy-preserving security, and compliance-aware cloud architecture. It is designed to support applications for research roles in security, privacy, applied cryptography, trustworthy AI systems, cloud security, and healthcare data protection.

## Research Focus

My research focuses on how privacy-preserving technologies, especially homomorphic encryption and advanced cloud-security controls, can protect sensitive healthcare data in cloud environments while balancing security, performance, scalability, and regulatory compliance requirements.

## Core Research Areas

- Homomorphic encryption and data-in-use protection
- Cloud-based healthcare databases and EHR security
- Privacy-preserving computing and applied cryptography
- HIPAA, GDPR, HITECH, NIST, and CSA-aligned compliance mapping
- Security, performance, and implementation trade-offs
- Cloud security architecture for regulated environments
- AI/cloud governance and trustworthy system design
